🤔 Domande Frequenti & Guida Tecnica

I tuoi dati sono criptati con AES-256. Senza la tua password, il file è inutilizzabile. Questo garantisce che se perdi la chiavetta USB o il file finisce in mani sbagliate, nessuno potrà vedere le tue transazioni.

Prima di fare test pericolosi, puoi copiare il file del database sul Desktop tramite il terminale di VS Code:

cp database.db ~/Desktop/database_backup_$(Get-Date -Format "yyyy-MM-dd").db

Questo comando crea una copia datata sul tuo Desktop. Per ripristinarla, basta chiudere Beesy e rimettere il file nella cartella del progetto.

  1. Scarica il backup da /backup-protetto.
  2. Aggiungi una transazione finta (es: "Test" 1€).
  3. Vai su /ripristino-protetto e carica il file.
  4. Se la transazione "Test" scompare e tornano i vecchi dati, il sistema è perfetto!
