Guida alla Sovranità dei Dati

🔐 Crittografia "Zero-Knowledge"

Beesy utilizza un sistema dove nemmeno il server conosce la tua Master Key in chiaro. Viene decriptata solo "al volo" nella memoria RAM quando inserisci la password. Se il server venisse compromesso, i tuoi backup resterebbero comunque illeggibili.

💾 Il file .json.enc

Il file scaricato contiene tutte le tue transazioni (Euro, Lightning, On-chain). Conservalo in un posto sicuro (come un password manager o una chiavetta USB criptata).

💡 Consiglio Pro: Verifica il tuo backup almeno una volta ogni tre mesi facendo un test di ripristino (dopo aver salvato il database attuale!).
